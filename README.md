# senzflow-sdk.js

senzflow sdk for node.js

## Install

`npm install senzflow-sdk.js --save`

## How to use

var Device = require("senzflow-sdk.js").Device;
var myDevice = new Device(options)

options

参数名            |类型              |说明
:----------------| :-----------  	| :-----------
url |String            |服务器地址 mqtts://senzflow.io:8883
